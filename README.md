# 💟 Itzagud Stats (ONYXScript)

A tool to handle infos on Itzagud.

<img width="392" height="844" alt="image" src="https://github.com/user-attachments/assets/dd67f9e9-7529-4227-a064-49e5494b4716" />


## ✨ Features
- **Small widget** that shows on all Itzagud site your **current Points/Clams**.
- Displays **Current Daily Wheel Spin amount** along with **Cooldown timer for next Daily** when daily limit reached.
- Displays **your remain time** till get **your upcoming 250 clams from chat!**
- Visible **toggle button to hide or unhide the widget.**
- Want to know your current winrate for Slots? Below shows **on-going Slots Games with your current entries and win rate** you need to check.
⠀
## 📋 How to use it?
- Download and install Tampermonkey/Violetmonkey/Greasemonkey.
- Open **itzagudstats.js** and it'll automatically **asks you to install on your previously chosen monkey**.
